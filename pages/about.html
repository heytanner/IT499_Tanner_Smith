About this prototype

This project demonstrates a responsive e-commerce website UI with simulated cart, checkout, and order tracking. It is built with static HTML, CSS, and JavaScript.

  • Browse products and add to cart
  • View and edit cart contents
  • Fake checkout flow with confirmation
  • Order tracking page using locally stored order data
  • Customer support chat UI
